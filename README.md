# Taller de Análisis de Bioimágenes (Train-the-Trainer)

This repository contains the organization and materials for the Train-the-Trainer course on Bioimage Analysis for LATAM (2026).

---

## How to Contribute / Cómo contribuir

### English
We welcome contributions to the course materials. Please follow these guidelines:
- **Slides & Presentations:** The goal is to publish all slides in Zenodo containing both the raw source files (`.tex` or `.ppt`) and the rendered `.pdf` files. The rendered view (PDF link) must be linked in the corresponding page in the `presentaciones` folder, along with its Zenodo citation.
- **Workshops & Practicals:** Workshops must be published in GitHub, using `pixi` tasks to set up the workspace. A release of the workshop should be published in Zenodo and cited in the workshop page. Inside the `practicos` folder, there should be a link pointing to the workshop repository on GitHub, the data (if necessary), and the Zenodo citation.

### Español
Agradecemos las contribuciones a los materiales del curso. Por favor, siga las siguientes pautas:
- **Diapositivas y Presentaciones:** El objetivo es publicar todas las diapositivas en Zenodo, incluyendo tanto los archivos fuente originales (`.tex` o `.ppt`) como los archivos renderizados `.pdf`. La vista renderizada (enlace al PDF) debe estar vinculada en la página correspondiente dentro de la carpeta `presentaciones`, junto con su correspondiente cita de Zenodo.
- **Talleres y Prácticos:** Los talleres deben publicarse en GitHub, utilizando tareas de `pixi` para configurar el espacio de trabajo. Se debe publicar una versión (release) del taller en Zenodo y citarla en la página del taller. Dentro de la carpeta `practicos`, debe haber un enlace que apunte al repositorio del taller en GitHub, los datos si fuera necesario, y la cita de Zenodo.
